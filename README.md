## Features
- No database required.
- Simple interface.
- Auto pairing with strangers.
- Skip button.
- Fast
- Store user's name in localSorage.

## Run your own

```bash
git clone https://github.com/shamilsaleem/ws-chat-app.git
cd ws-chat-app
npm install
npm start
```
